# SBAN Supporter
模倣に役立つツール集です。
## ツール一覧
- num_to_midi<br>
  バイナリデータのテキストをMIDIに変換する関数(36進数まで対応してます)
- morse_to_midi<br>
  モールス信号のテキストをMIDIに変換する関数
- tenji_to_midi<br>
  点字のテキストをMIDIに変換する関数
- midi_to_image<br>
  MIDIを画像に変換する関数
- midi_to_num<br>
  MIDIから音の高さだけを抜き出す関数
- make_hiragana_small<br>
  AviUtlでひらがなを制御文字で小さくする関数(「漢字以外を小さくする」という方が正しい)
